Valorant is a free-to-play multiplayer first - person shooter developed by Riot Games
It is the first game that Riot Games has developed in this genre. The game was first announced in October 2019, codenamed Project A
It was released for Windows on June 2, 2020
The release was originally planned for summer 2020
A closed beta version started on April 7, 2020
Participation in the closed beta
was achieved by sending game keys for watching live streams of the game on the Twitch video platform
The Twitch account had to be linked to the Riot Games account
On June 2, 2020, Valorant left the beta phase
